# Pokémon vermell i blau (Projecte de localització al català)

Aquest és un desassemblatge de Pokémon vermell i blau basat en [Pokered](https://github.com/pret/pokered/).

El projecte de localització de Pokémon Edició vermella i Edició blava conté els diàlegs originals de l'edició japonesa vermella/verda, així com els noms localitzats dels 151 Pokémon, pobles i entrenadors. A més a més, la Pokédex té el sistema mètric restaurat i les descripcions originals dels 151 Pokémon.
